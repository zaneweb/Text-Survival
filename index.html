Text Survival

Nothin like a non-graphical game.

Text Survival is a minecraft like game, but with no graphics.

Download Guide

Downloading / installing Text Survival is very easy

  • Step one
    • Download the latest python 2.x.x version from python.org
  • Step two
    • Clone Text survival with the following: git clone https://github.com/Draik09/A-text-game.git
    • Or you can download the zip / tar.gz file from the github pages
  • Step three
    • Go into the Text-Survival folder and launch the textGame.py file with the python launcher.
  • Step four
    • Enjoy!
